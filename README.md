# NeetCode Solutions

This repository contains my accepted solutions to problems from the **NeetCode** roadmap. It serves as a record of my DSA practice and reflects my ongoing preparation for software engineering interviews and competitive programming.

The repository is automatically synchronized with NeetCode, so every accepted submission is pushed here and my progress is tracked over time.

---

## About

* Solving problems from the NeetCode roadmap.
* Solutions are primarily written in **C++**.
* Automatically synced from NeetCode after every accepted submission.
* Focused on improving problem-solving skills, algorithmic thinking, and coding efficiency.

---

## Repository Structure

Solutions are organized by topic and problem name. If a problem has been submitted multiple times, each accepted submission is stored separately.

```text
<topic-folder>/
  <problem-name>/
    submission-0.<ext>
    submission-1.<ext>
    ...
```

### Example

```text
Data Structures & Algorithms/
├── binary-search/
│   └── submission-0.cpp
├── valid-anagram/
│   └── submission-0.cpp
└── two-sum/
    ├── submission-0.cpp
    └── submission-1.cpp
```
